RT-Thread是一个来自中国的开源物联网操作系统，它提供了非常强的可伸缩能力：从一个可以运行在ARM Cortex-M0芯片上的极小内核，到中等的ARM Cortex-M3/4/7系统，甚至是运行于MIPS32、ARM Cortex-A系列处理器上功能丰富系统。

## 支持的芯片架构 ##

RT-Thread支持数种芯片体系架构，已经覆盖当前应用中的主流体系架构：

* ARM Cortex-M0
* ARM Cortex-M3/M4/7
* ARM Cortex-R4
